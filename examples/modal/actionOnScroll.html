Configuration overview and validation scenarios for actionOnScroll. This plugin tracks document scroll distance and runs modal actions (open, close, destroy) after configured amount thresholds.

ActionOnScroll Instances

Each checkbox arms one scenario. Select tests, refresh the page, then scroll downward by the configured amount. Use the scroll target at the bottom to re-test longer distances.
