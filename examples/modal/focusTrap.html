Modal | focusTrap

Configuration overview and validation scenarios for focusTrap. This plugin keeps keyboard focus inside the modal panel, optionally focuses the first focusable control on open, and restores focus after close/destroy.

FocusTrap Instances

Each card opens one validation scenario of the focusTrap plugin and shows the exact configuration in a code window. Use Tab, Shift+Tab, Escape, and the visible controls inside each modal to verify focus behavior.
